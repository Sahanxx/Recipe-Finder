React + Vite + Tailwind. Search recipes from TheMealDB, view details, add favorites (localStorage).
- Debounced search
- Favorites with toasts
- Dark/Light theme toggle
- Skeleton cards
- Infinite scroll

## Run
npm install
npm run dev

## Build
npm run build
npm run preview
